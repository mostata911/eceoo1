Definition of the operating system


It is a group of software responsible for managing resources and computer software and it represents an intermediary between the user and computer hardware. In another definition the operating system represents a bridge for operating user programs and performs the basic tasks



system definition examples
Single-user single-task The simplest type of operating system serves one user at a time and is a single task DOS-MS
Single user multi-tasking This system has the ability to run more than one program simultaneously, as the CPU goes Windows95, Windows98
Multiplayer single user task This system allows a number of people to implement one program at a time. Windows NT
Multiuser multitasking It is a modern operating system that allows the user to run several programs even if there are multiple processors unix
