Operating system components




    1. Kernel

    • The operating system kernel provides the first level to control all computer pursuits

    2. Process Execution

    • The operating system provides an interface for additional parts with an application program in order to enable the programs to communicate with the device by following the principles and procedures

    3. Interrupt

    • Interrupts are necessary because they give the operating system a reliable technique to interact with their environment

    4. Memory Management

    • The mission of the operating system is to manage the main memory and memory, and also move the processes back and forth between the disk and the main memory during execution

    5. Multitasking

    • Many independent computer programs describe the work that allows multitasking at the operating system of a user to perform a single task

    6. Networking

    • Networks are the interaction of a processor through communication steps. The communication network must take into account safety guidance, communication methods, and security problems

    7. Security

    • It protects operations and depends on a variety of technologies and can be obtained to run the program on the system

    8. User Interface

    • Part of the operating system allows the operator to obtain user interface information and commands that are written via the command line via the keyboard
